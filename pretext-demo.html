Pretext + Canvas 2D

Animating text like cherry blossoms

Each line of text is a petal. It drifts, sways, and settles — landing exactly where Pretext calculated it should, with no DOM reads in the animation loop.

Poem lines as falling petals

prepareWithSegments + layoutWithLines compute each line's exact target position once. The animation loop uses only those cached coordinates — zero DOM reads, zero reflow.

Museum label with exact card height

Pretext measures the body text at a fixed column width and returns the precise line count. The card border is drawn from that number — it always fits the text, at any font size or locale.
